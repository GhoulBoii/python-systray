<h1 align=center>Windows Systray Viewer</h1>

<h3 align=center>Python Application to view Windows System Tray icons.</h3>

## Dependencies

- pillow
- pywin32

## Installation

- Install python
- Install dependencies
    - `pip install -r requirements.txt`
- Run script
    - `python src/main.py`
